# Chemistry boundaries

Stop and request targeted confirmation for uncertain regioisomers, tautomers, salts, solvates, protonation, stereochemistry, charge, multiplicity, coordination mode, or metal basis/ECP choices.

For ordinary closed-shell main-group organic molecules, use the formal charge in the structure and singlet multiplicity only when no radical electrons are present. Treat B3LYP/6-31G(d) as a preview-compatible placeholder route, not a universal scientific recommendation.

Do not infer axial chirality from an empty tetrahedral-center list. Retain the original ChemDraw preview and manifest for BINOL/BINAP/SPINOL and related atropisomeric structures.

Automatic ETKDGv3 plus MMFF94s/UFF geometry is a preview starting point, not a validated conformer search or protocol for metal complexes, transition states, excited states, ion pairs, or unusual bonding.
