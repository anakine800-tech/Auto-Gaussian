# Auto-G16 v2.5 cross-Skill integration contract

`gaussian-v25-integration-review/1` is an offline owner overlay. It replays an
exact method-evidence brief, explicit human method decision, selected closure
plan, per-route TS-seed portfolio, immutable execution-batch review, and its
persistent ledger. It changes none of the owner schemas.

For every selected closure route, the mechanism and user-confirmation gates
must cite the exact mechanism discussion; the method gate must cite both the
brief and the discussion; and `initial_guess_quality` must cite the exact
portfolio. Every calculation-bearing selected node maps one-to-one to the
review and ledger task set, which remains capped at ten distinct scientific
identities. The TS/Freq task additionally binds the primary seed geometry.

The overlay fixes `calculation_ready: false`, `executable: false`,
`no_submission_authorization: true`, `no_automatic_promotion: true`, and
`live_actions: false`. Later-stage dependency evidence, exact input review,
and a fresh live approval for each physical attempt remain separate gates.
See the repository release-facing explanation in
`docs/v2.5-integration-approval.md`.
