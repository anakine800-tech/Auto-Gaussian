# 零基础学习路线

## GKB-0001 计算化学在这里指什么

- 类型：概念
- 关键词：计算化学；computational chemistry；量子化学；分子模拟
- 零基础解释：用计算机中的物理模型研究分子的结构、能量、性质和反应。量子化学计算求解或近似求解量子力学方程；分子模拟还包括分子力学和分子动力学。
- 严格含义：计算化学是理论化学、计算机科学与化学的交叉领域，常见范围包括量子化学、分子力学和分子动力学。
- 不能据此推出：计算结果能替代实验，或所有化学问题都适合 Gaussian。

## GKB-0002 学习知识主线

- 类型：学习路径
- 关键词：学习顺序；知识体系；Gaussian beginner
- 建议主线：计算方法 → 基组 → Gaussian/GaussView → 优化、频率和光谱等作业类型 → 研究实例 → 阅读与写作计算化学论文。
- 零基础使用：先知道“为什么算”，再理解“用什么模型”，然后学习“输入如何写、输出怎么看”，最后才进入过渡态、IRC、溶剂和研究机理。

## GKB-0003 一项计算研究的四步

- 类型：研究 SOP
- 关键词：研究流程；计算目的；方法选择；结果分析；误差评价
- 步骤：
  1. 明确问题：结构、性质、反应机理、规律或实验现象中的哪一项需要解释。
  2. 选择模型和方法：决定使用量子力学、分子力学或其它模型，并匹配研究体系和资源。
  3. 执行计算：例如优化、构象搜索、动力学或具体性质计算。
  4. 分析与评价：提取性质，检查计算是否实施正确，并评价误差和结论边界。
- 成功证据：问题、方法、计算步骤和结论之间有明确对应关系，而不只是得到一个正常结束的输出文件。

## GKB-0004 面向 Gaussian 的建议学习阶段

- 类型：学习路径
- 关键词：beginner roadmap；入门；练习顺序
- 阶段 1：术语和证据——能量、结构、势能面、稳定点、过渡态、频率、热化学。
- 阶段 2：理论选择——HF、DFT、后 HF 方法的含义与成本；基组、极化、弥散、ECP。
- 阶段 3：输入输出——五段输入结构、电荷与多重度、坐标、route section、检查点文件、输出终止和结果位置。
- 阶段 4：稳定结构——合理初始结构、优化、频率和热化学。
- 阶段 5：反应研究——构象覆盖、TS、虚频模式、IRC、共同参考态和能垒。
- 阶段 6：性质与环境——溶剂、弱相互作用、NMR、激发态和相对论处理。
- 阶段 7：研究闭环——文献依据、模型假设、数据整理、图表和可复现记录。
- 说明：这是为零基础学习整理的建议顺序，不是唯一固定路线。

## GKB-0005 学习和操作习惯

- 类型：操作原则
- 关键词：记录；监控；实践；文献
- 建议：重视上机练习；边学边用；多读书和文献；记录方法、输入、版本和中间决策；计算运行中持续检查进程与结果。
- 常见误区：只收集关键词和模板，不理解适用条件；任务提交后直到结束才查看；只保存最终数值而不保存输入输出。

## 入门回答的默认检查问题

回答任何实际 Gaussian 问题前，先确认：

1. 研究目标是什么？
2. 分子连接、立体化学、电荷和多重度是否确定？
3. 当前步骤是结构准备、优化、频率、单点、TS、IRC 还是性质计算？
4. 方法和基组为什么适合该体系？
5. 哪些输出证据决定成功？
6. 哪些结论需要实验、文献或更高层级计算验证？
